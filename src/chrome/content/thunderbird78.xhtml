Clippings for Thunderbird 6.0

Clippings for Thunderbird 6.0, which is compatible with Thunderbird 78, is now available.

What’s New

  • The Clippings Manager window and the New Clipping dialog have a similar look and feel as the same found in Clippings for Firefox
  • It’s once again possible to drag and drop clippings and folders in the clippings tree list in Clippings Manager
  • In the New Clipping dialog, you can preview the clipping when applying the options to create as unquoted text or remove extra line breaks

How to Get It

Clippings for Thunderbird 6.0 is not compatible with your current version of Thunderbird, so you’ll need to update to Thunderbird 78 first.

Once Thunderbird is updated to version 78, Clippings should be automatically updated. Double-check that the update was successful by opening Add-ons Manager (from the Thunderbird menu (≡), click Add-ons), and look for Clippings in the list of installed add-ons.

Using Sync Clippings?

Install the Sync Clippings Helper, a background program that saves your synced clippings to a file on your system. Available for download from the AE Creations website.

Compatibility With Other Versions of Thunderbird

Clippings for Thunderbird 6.0 is compatible with Thunderbird 78 only. It is not compatible with Thunderbird 68. If you plan on staying with Thunderbird 68, you’ll need to continue using Clippings for Thunderbird version 5.7.
