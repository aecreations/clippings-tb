Coming soon!

Compatibility With Thunderbird 78

Clippings for Thunderbird 6.0 is coming soon, and will be compatible with Thunderbird 78.

What's New

Here are the new features and improvements coming in Clippings for Thunderbird 6.0:

  • The Clippings Manager window and the New Clipping dialog have a similar look and feel as the same found in Clippings for Firefox
  • It's once again possible to drag and drop clippings and folders in the clippings tree list in Clippings Manager
  • In the New Clipping dialog, you can preview the clipping when applying the options to create as unquoted text or remove extra line breaks

Stay Informed

Once Clippings for Thunderbird 6.0 is released, an announcement will be posted to the AE Creations blog (Atom feed) and the AE Creations Help and Support Forums.

After you've upgraded Thunderbird to version 78, make sure to also update Clippings to version 6.0.

Using Sync Clippings?

After installing Clippings for Thunderbird 6.0, you'll need to install Sync Clippings Helper version 1.1, which will be available for download from the AE Creations website.

Compatibility With Other Versions of Thunderbird

Clippings for Thunderbird 6.0 will be compatible with Thunderbird 78 only. It will not be compatible with Thunderbird 68. If you plan on staying with Thunderbird 68, you'll need to continue using Clippings for Thunderbird version 5.7.
